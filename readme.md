# Workshop de Machine Learning  

Obrigado por participares neste workshop das JEF, organizado pelo NFIST.

Este workshop foi preparado pelo **noisr**, um grupo de trabalho premiado do Departamento de Física do Instituto Superior Técnico. Nós desenvolvemos **redes neurais físicas** que aprendem sem serem programamada, apenas através da própria evolução natural do sistema físico.

Temos um laboratório no Departamento de Física onde desenvolvemos o nosso protótipo e pretendemos que este complemente as arquiteturas de computadores e placas gráficas, para uma computação mais rápida e energeticamente mais eficiente.

Estamos a recrutar pessoas talentosas, de diferentes cursos e backgrounds, interessadas em integrar a nossa equipa. Se estiveres interessado/a, podes contactar-nos via instagram **@noisr.ist**
